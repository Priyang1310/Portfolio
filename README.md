# Portfolio Website

## Overview
This is my personal portfolio website built to showcase my projects and skills. The site is developed using React.js, Three.js, and Framer Motion to create a dynamic and engaging user experience. EmailJS is used for the contact form to enable visitors to get in touch with me directly through the website.

## Technologies Used
- **React.js**: A JavaScript library for building user interfaces
- **Three.js**: A JavaScript library for 3D graphics on the web
- **Framer Motion**: A library for animations in React
- **EmailJS**: A service to send emails directly from the client-side

## Features
- Interactive 3D elements and animations
- Responsive design for mobile and desktop
- Contact form powered by EmailJS
